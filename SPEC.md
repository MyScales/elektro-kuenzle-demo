# Putztech GmbH - Website Redesign Demo

## Business Info
- **Name:** Putztech GmbH
- **Industry:** Gipser, Trockenbau, Fassadenarbeiten
- **Location:** Neuhausen am Rheinfall / Schaffhausen Region
- **Current Website:** putztech.ch (very minimal, needs upgrade)

## Design Requirements

### Vibe
- **Swiss minimalist** - clean, lots of whitespace
- **Modern & premium** - not cheap template look
- **Trust-building** - established, professional
- **NOT typical AI-generated** - unique, handcrafted feel

### WOW Effects (must have)
1. **Smooth scroll animations** - elements fade/slide in on scroll
2. **Parallax backgrounds** - subtle depth effect
3. **Hover micro-interactions** - buttons, cards respond
4. **Hero video/image with overlay text animation**
5. **Before/After slider** for renovation projects (use placeholder images)

### Color Palette
- Primary: Deep charcoal (#1a1a1a) or dark blue (#0f172a)
- Accent: Warm orange/gold (#f59e0b) or teal (#0d9488)
- Background: Off-white (#fafafa)
- Clean contrasts, professional feel

### Sections (One-Pager)
1. **Hero** - Bold headline, subline, CTA button, background image
2. **Services** - 3 cards (Innenputz, Aussenputz/Fassade, Trockenbau)
3. **Why Us** - 3-4 trust points (Erfahrung, Qualität, Regional, Zuverlässig)
4. **Projects/Gallery** - Before/After slider or image grid (placeholder)
5. **Testimonials** - 2-3 fake but realistic quotes
6. **Contact** - Simple form + address + phone + map placeholder
7. **Footer** - minimal

### Tech Stack
- Pure HTML + CSS + vanilla JavaScript (no frameworks needed)
- GSAP or AOS for scroll animations
- Mobile-first, fully responsive
- Single index.html file (easy to demo)
- CSS in <style> tag or separate file
- JS in <script> tag or separate file

### Content (German)
- Headline: "Verputzt. Perfekt." or "Qualität, die man sieht."
- Subline: "Ihr Spezialist für Gipserarbeiten in der Region Schaffhausen"
- Services:
  - Innenputz: Wände und Decken perfekt verputzt
  - Fassade: Witterungsbeständig und schön
  - Trockenbau: Flexible Raumlösungen
- CTA: "Jetzt Offerte anfragen"

### Files to Create
- index.html (main file)
- styles.css (optional, can be inline)
- script.js (optional, can be inline)
- /images folder with placeholder instructions

## Quality Checklist
- [ ] Looks premium, not template-y
- [ ] Animations are smooth, not jarring
- [ ] Mobile responsive
- [ ] Fast loading
- [ ] Clear CTA
- [ ] Would make business owner say "WOW, I want this!"
